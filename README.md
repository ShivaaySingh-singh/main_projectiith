#  SRC ERP System

### Enterprise Resource Planning System  - Sponsored Researhc & Consultancy service

# Overview

A secure, role-based ERP system built exclusively fro an organization to streamline the complete lifecycle of research project funding - from project creation to final payment disbursement.
Faculty members can securely login to track their project funds and billing records,
while the R&D admin team gets a powerful dashboard to manage evrything end-to- end.

## 👥 User Roles

### 🎓 Faculty Portal
- Secure login with personal credentials (ID + Password)
- View **only their own** assigned research projects
- Track **project fund utilization** in real-time
- Monitor **billing records** and payment status
- Complete data isolation — no faculty can see another's data

### 🛠️ Admin Panel (SRC R&D Team)
- Full control dashboard for the working team
- Project creation and lifecycle management
- Payment synchronization across all projects
- Bill creation, assignment & tracking
- Assign billing tasks between admin members
- Faculty account creation & management
- End-to-end oversight from project initiation → payment completion

## ✨ Key Modules

| Module | Description |
|--------|-------------|
| 🔐 Authentication | Role-based secure login (Faculty / Admin) |
| 📁 Project Management | Create, assign, and track research projects |
| 💰 Fund Tracking | Real-time fund utilization per project |
| 🧾 Billing System | Bill creation, assignment & status tracking |
| 💳 Payment Sync | Synchronize payments across projects & members |
| 👤 Faculty Management | Admin can create & manage faculty accounts |
| 📊 Admin Dashboard | Full overview panel for Admin team |
| 🔒 Data Isolation | Faculty sees only their own project data
